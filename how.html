Getting started
• Create or join a room using the "Create / Join Room" button.
• Pick a nickname and color. Guests are auto-assigned a grey color.
Rooms
• Rooms list is on the left (desktop) or in the drawer (mobile).
• Use the lobby size control to change left panel width.
Sending messages
• Type a message and press Enter to send.
• Use the emoji picker (button next to input) to insert emojis.
Replying
• Desktop: Hover a message and click "reply" to reply to it.
• Mobile: Swipe a message to the right (short swipe reveals reply preview) to start a reply; or tap the reply button.
Edit & Delete
• Desktop: Your messages have a three-dot "more" button at the far right — open it for Edit and Delete.
• Mobile: Long-press your message to open the Edit/Delete menu.
Typing & Seen receipts
• Typing indicator shows who is currently typing in the room.
• Hover a message (desktop) to see a tooltip with users who have seen it.
Keyboard shortcuts (desktop)
• Esc → Close menus
• Ctrl/Cmd + Enter → Send message
• ? → Open “How To”
Mobile specifics
• Drawer: swipe from the left edge or use the blue button to open the room drawer.
• On-screen keyboard: input and reply box move to stay visible; the chat will auto-scroll when appropriate.
• Swipe-to-reply now moves the *entire message element* so the reply gesture feels direct and intuitive.
Desktop specifics
• Lobby and chat sidebar remain visible; three-dot menus are available for edits.
• Hover interactions show profile and seen receipts.
Account notes
• Login/Register stores credentials locally (for demo). Guests are labeled "Guest####".
• Settings gear lets you logout or delete your account (local storage + optional cloud doc removal).
